## Phase 1 — Foundation ✅ (ШАГ 1, 2026-06-05)

Каркас на Rust + objc2 готов и проверен вживую на Tahoe. В строке меню живут ДВА статус-айтема: видимый якорь `<` (кликом прячет/возвращает иконки) и спейсер-cutter, который раздувается до ~ширины экрана и выталкивает иконки левее себя за край (путь HiddenBar, без хаков).

Главный урок ШАГА 1: порядок айтемов на Tahoe НЕ задаётся порядком создания, и нет API двигать их программно. Поэтому скрытие защищено guard'ом — раздуваем спейсер только если он реально левее якоря, иначе блокируем (`⚠<`). Это убрало баг «`<` улетел за край». Подробности — ADR 005 (поправка) и research journal 003.

Спринты `01_CORE_status-item-ffi`, `02_CORE_hide-show` — завершены, в `sprints/archive/`.

## Phase 2 — Second row UI

Полу-IceBar: полоска-окно (`NSPanel`) под строкой меню со своими ярлыками-приложениями, которые пользователь сам туда кладёт. Не зеркало чужих иконок, без Screen Recording/Accessibility. Спринт `03_UI_second-row`.

## Phase 3 — Polish

Автоскрытие по таймеру, настройки с hotkey и автозапуском, современная иконка и визуальный стиль. Спринты `04_CORE_autohide-timer`, `05_CORE_settings-hotkey-autostart`, `06_UI_icon-branding`.
