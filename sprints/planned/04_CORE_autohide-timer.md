# 04 CORE — Автоскрытие по таймеру

Дата создания: 2026-06-05

## Цель

Раскрытое состояние прячется само после периода бездействия: полоска полу-IceBar закрывается, а выдвинутые спейсером иконки снова уезжают за край.

## Объём работ

- [ ] Таймер бездействия для раскрытого состояния (открытая полоска + показанные иконки).
- [ ] Сброс таймера при наведении/взаимодействии с полоской.
- [ ] По истечении таймера: закрыть полоску полу-IceBar и свернуть спейсер обратно (иконки за край).
- [ ] Настраиваемая задержка (значение по умолчанию + хранение в настройках).

## Критерий готовности

После заданной паузы бездействия полоска закрывается, а иконки снова прячутся спейсером за край; взаимодействие с полоской сбрасывает таймер.
