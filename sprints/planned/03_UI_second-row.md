# 03 UI — Второй ряд (полу-IceBar)

Дата создания: 2026-06-05

## Цель

Сделать «второй ряд» — полоску-окно под строкой меню, в которой живут СВОИ ярлыки пользователя (приложения и действия, которые он сам туда положил). Это полу-IceBar: полоска-лаунчер своих плашек, а НЕ зеркало чужих системных иконок. Открывается по клику на разделитель `<`.

> Явно: без Screen Recording, без синтетических `CGEvent`, без Accessibility. Это просто своё окно со своим контентом (см. ADR 004). Полный Ice (снимки чужих иконок) — отвергнут.

## Объём работ

- [ ] Borderless `NSPanel`: `styleMask [.nonactivatingPanel, .fullSizeContentView, .borderless]`.
- [ ] `level = .mainMenu + 1`, `collectionBehavior [.fullScreenAuxiliary, .ignoresCycle, .moveToActiveSpace]`.
- [ ] Позиционировать под строкой меню: `originY = screen.maxY - 1 - menuBarHeight - frameHeight`.
- [ ] Наполнить полоску своими кнопками-ярлыками (плашки приложений/действий).
- [ ] Хранить список ярлыков в настройках; пользователь сам добавляет/убирает приложения (выбор через open-panel).
- [ ] Клик по ярлыку — запуск приложения/действия.
- [ ] Показ/скрытие полоски по клику на разделитель `<`.
- [ ] Корректное поведение на экране с чёлкой и при смене разрешения/монитора.

## Критерий готовности

Клик по разделителю открывает полоску под строкой меню; в ней видны свои ярлыки, кнопки кликабельны и запускают приложение/действие; повторный клик полоску закрывает. На экране с чёлкой полоска видна и не перекрывается.
