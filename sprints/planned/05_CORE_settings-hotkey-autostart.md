# 05 CORE — Настройки, hotkey, автозапуск

Дата создания: 2026-06-05

## Цель

Дать пользователю настройки: горячую клавишу для переключения скрытия и автозапуск при входе в систему.

## Объём работ

- [ ] Окно настроек.
- [ ] Глобальный hotkey для toggle (скрыть/показать).
- [ ] Автозапуск при логине через `ServiceManagement` / `SMAppService`.
- [ ] Хранение настроек между запусками (задержка автоскрытия из спринта 04, список ярлыков полу-IceBar из спринта 03).

## Критерий готовности

В окне настроек можно задать hotkey и включить автозапуск; hotkey переключает скрытие; после перезагрузки приложение само запускается, настройки сохранены.
